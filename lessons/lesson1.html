ECMAScript

Sounds like a skin disease?

ECMAScript is the name of the language specification. Javascript in an implementaion of ECMAScript

More precisely, Javascript is a collective name for a set of ECMAScript implementations called 'engines'

Javascript Engines

  • V8 - Chrome
  • SpiderMonkey - Firefox
  • Nitro - Safari
  • Carakan - Opera (no longer developed)
  • Chakra - IE9+

Node.js

Node.js is an environment for writing programs with evented, asynchronous I/O. Does that clear everything up?

Node.js = the V8 javascript engine + an asynchronous I/O library called libuv + a few other goodies

Node.js

For our purposes right now. Node.js is an environment for writing vanilla flavor javascript outside of a browser environment.

Let's install it on our computers!

Welcome to the REPL!

  • R - Read
  • E - Eval
  • P - Print
  • L - Loop

This means that everytime you type something and press 'Enter', the interpreter will read your input, evaluate it (including executing any instructions it contains), print the result and looparound to prepare for your next instruction

Expressions

Expressions are commands that collectively evaluate to some value. When we type in an expression, the REPL returns the collective value of the expression we typed

Let's try this out with some mathematical expressions.


2
// 2
1 + 10
// 11
2 + 4 * 2
// 10
(2 + 4) * 2
// 12

Operators

Operators allow us to build complex expressions out of simple ones. Javascript has a lot of operators. When the interpreter begins to evaluate an expression, its operators are processed in a particular ranking order called operator precedence.Understanding operator precedence is really important, but you don't have to absorb the entire chart right now

Variables

Variables in javascript are reference names where we can store expression values for later use. Variables are created (declared) with a variable declaration statement


var aValue = 1;

var anotherValue = 2;

aValue + anotherValue;
// 3

Variable Types

Javascript has a few different types of variables, and we'll be getting to know them all intimately during this course. For now, though, we only need to deal with a couple types: Strings and Numbers


// This is an example of working with strings
var firstName = "David";
var lastName = "Manning";

firstName + lastName;
